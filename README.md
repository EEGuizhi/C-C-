## Learning：
- <b>ASCII TABLE</b>：<br>
    (char) 'A' = (int) 65 <br>
    (char) 'a' = (int) 97 <br>
    (char) '0' = (int) 48 <br>

- <b>Data type range</b>：<br>
    int (32 bits)： +-( 2^(31) - 1 ) <br>
    unsigned int (32 bits)： 0 ~ ( 2^(32) - 1 ) <br>
    long long int (64 bits)： +-( 2^(63) -1 ) <br>
